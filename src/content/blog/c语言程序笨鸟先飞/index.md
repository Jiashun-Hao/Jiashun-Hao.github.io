题目描述  
lin2144是一只小菜鸟，都是笨鸟先飞，lin2144想来个菜鸟先飞，他从0点出发  
一开始的飞行速度为1m/s，每过一个单位时间lin2144的飞行速度比上一个单位时间的飞行速度快2m/s，问n (0 < n < 10^5)个单位时间之后lin2144飞了多远?

输入  
输入一个T表示为有几组数据  
每组数据输入一个n，表示lin2144飞行的时间.

输出  
输出lin2144飞行了多远，因为数字很大，所以对10000取模.

——————————————————我是分割线——————————————————————————  
解析，这道题刚刚拿到的时候有点模糊的概念，所以建议初学者用笔在纸上列写一些东西；  
如 ；时间 、速度、已走路程  
这样很清楚可以知道时间与路程的关系结果为天数的平方，第一天 1米 第二天 4米 第三天 9米…

---

知道此概念之后再考虑与10000取模的事情，这时就会出现两种情况——计算前取模与计算后取模;计算后取模毫无疑问会使数据溢出，故选择前者。解析完毕；  
附上代码：

```
#include <stdio.h>
#include <stdlib.h>

int main()
{
    int t,n;
    scanf("%d",&t);
    while(t--)
    {
        scanf("%d",&n);
        printf("%d\n",n*n%10000);
    }
    return 0;
}
```
